# rrun
Remote script executor

## Install

```bash
curl -sSL https://raw.githubusercontent.com/ValentinKolb/rrun/main/rrun.sh | sudo tee /usr/local/bin/rrun > /dev/null && sudo chmod +x /usr/local/bin/rrun
```

## Usage

```bash
rrun -u  https://raw.githubusercontent.com/ValentinKolb/rrun/main test.sh
```
